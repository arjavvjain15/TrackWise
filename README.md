# TrackWise - MERN Expense Tracker

TrackWise is a full-featured expense tracking web application built using the MERN stack (MongoDB, Express.js, React.js, Node.js). It helps users seamlessly track, manage, and visualize their daily expenses. With a clean and responsive UI powered by Tailwind CSS, TrackWise offers a smooth experience for budgeting and analyzing spending habits — perfect for anyone aiming to take control of their personal finances.
